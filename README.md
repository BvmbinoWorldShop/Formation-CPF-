## 🚀 Mettre en ligne sur GitHub Pages (gratuit, permanent)

### 1. Créer un compte GitHub
→ github.com → Sign up (gratuit)

### 2. Créer un repository
- Clique **"New"** (bouton vert)
- Nom : `formationpro`
- Coche **"Public"**
- Clique **"Create repository"**

### 3. Upload le fichier
- Dans le repo, clique **"uploading an existing file"**
- **IMPORTANT** : Renomme `FormationPro.html` en `index.html` avant d'uploader
- Glisse le fichier → Clique **"Commit changes"**

### 4. Activer GitHub Pages
- Onglet **Settings** → menu gauche **Pages**
- Source : **"Deploy from a branch"**
- Branch : **main** · Folder : **/ (root)**
- Clique **Save**
- Attends 2-3 minutes

### 5. Ton URL permanente
```
https://TON-USERNAME.github.io/formationpro
```

## 📱 Installer sur iPhone (mode app native)

1. Ouvre l'URL dans **Safari** (pas Chrome !)
2. Icône **Partager** (carré + flèche en bas)
3. Sélectionne **"Sur l'écran d'accueil"**
4. Nomme-la **FormationPro**
5. **Ajouter**

→ L'app s'ouvre en plein écran, sans barre Safari.
→ Fonctionne comme une vraie app native.
